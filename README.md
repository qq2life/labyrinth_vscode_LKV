# Лабиринт сокровищ

Учебный текстовый квест, соответствующий критериям оценки. Реализованы базовые и продвинутые механики, условия победы, структура проекта, линтинг и Makefile.

## Авторы
- Kirill Lozo <lozo-kirill@yandex.ru>

## Установка и запуск

### Требования
- Python 3.10–3.12
- [Poetry](https://python-poetry.org/)

### Установка
```bash
make install
```

### Запуск игры
```bash
make project
```
или
```bash
poetry run labyrinth
```

### Линтинг
```bash
make lint   # проверка
make fix    # автоисправление + форматирование
```

## Команды в игре
- `go <направление>` — перемещение (север/юг/восток/запад; можно писать русские слова)
- `look` — осмотреться
- `inventory` — инвентарь
- `take <предмет>` — взять предмет
- `use <предмет> [на_объект]` — использовать предмет (например, ключ в Сокровищнице)
- `solve <загадка> <ответ>` — решить загадку (в Пазл-холле: `solve сфинкс человек`)
- `help` — помощь
- `quit` — выход

## Цель игры
Откройте сундук в **Сокровищнице** с помощью **золотого ключа**, который выдаётся за решение загадки в **Пазл-холле**. По пути возможны ловушки (дротики/яма) и случайные события; некоторые предметы (амулет, щит) помогают.

## Архитектура
- `constants.py` — карта, предметы, загадки, тексты и команды
- `utils.py` — утилиты вывода и парсинга
- `player_actions.py` — логика действий и механики (в т.ч. ловушки/события)
- `main.py` — игровой цикл и маршрутизация команд
